# Prompt Management: Science Coach Agent

This directory contains all prompts used by the science coach agent feature (002-science-coach-agent).

## Structure

- `bjj-detection-prompt.md`: Prompt for classifying user messages as BJJ-related or not
- `sports-science-coach-prompt.md`: Prompt for generating sports science explanations

## Version Control

All prompts are version controlled and include metadata headers:
- Purpose
- Target agent
- Version
- Last modified date
- Usage context

## Updates

When updating prompts:
1. Increment version number
2. Update last modified date
3. Document changes in commit message
4. Test prompt changes before deploying

## Constitution Compliance

This structure complies with Principle 6 (Agent Prompt Management) of the project constitution, ensuring prompts are:
- Stored separately from code
- Version controlled
- Easy to iterate and improve
- Reviewable and auditable
